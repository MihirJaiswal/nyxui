## About

Nyx UI is a modern, open-source React component registry built for Next.js. It provides a collection of customizable, animated, and accessible UI components designed to help developers build stunning interfaces faster.

Built with [Tailwind CSS](https://tailwindcss.com/), [Motion](https://www.motion.dev/), and [TypeScript](https://www.typescriptlang.org/).

## Features

- **Pre-built Components** - A growing collection of production-ready components
- **Animations** - Smooth, customizable animations powered by Framer Motion
- **TypeScript** - Fully typed for excellent developer experience
- **Tailwind CSS** - Styled with utility-first CSS framework
- **Accessible** - Built with accessibility in mind using Radix UI primitives
- **shadcn Compatible** - Works seamlessly with the shadcn/ui ecosystem

## Quick Start

### Prerequisites

- Node.js 18+
- pnpm (recommended)

## Usage

Browse components at [nyxui.com](https://nyxui.com) and copy the code directly into your project. Each component is designed to be self-contained and easily customizable.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Content**: [MDX](https://mdxjs.com/) with Content Collections
